## 🧪 Jak testować

1. Zainstaluj plugin [Skript](https://github.com/SkriptLang/Skript)
2. Wrzuć pliki `.sk` do `plugins/Skript/scripts`
3. W grze wpisz `/sk reload <nazwa>`
